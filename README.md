# Vas-y

## Usage
```
$ vasy simple
cd ~/Projects/simple/
```


## Install

```
./install.sh [~/.bash_profile]
```

## .vasy file
```vasy``` looks into `~/.vasy` file which is formatted with `key: path`.

Example:
```
app-1: ~/Projects/app1
app-2: ~/Projects/app2
simple: ~/Projects/simple/
```


## Notes
```vasy``` comes from *vas-y* French for *go on*
